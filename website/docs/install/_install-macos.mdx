### Download

The macOS versions of **xPack OpenOCD**
are packed as `.tar.gz` archives.
Download the latest version named like:

* <code>xpack-openocd-{ props.upstreamVersion }-{ props.xpackSubversion }-darwin-x64.tar.gz</code>
* <code>xpack-openocd-{ props.upstreamVersion }-{ props.xpackSubversion }-darwin-arm64.tar.gz</code>

:::note

In case you wonder where the suffix comes
from, it is exactly the Node.js `process.platform` and `process.arch`.

:::

### Unpack

To manually install the <b>xPack OpenOCD</b>,
unpack the archive and move it to <code>~/.local/xPacks/openocd/xpack-openocd-{ props.upstreamVersion }-{ props.xpackSubversion }</code>:

<CodeBlock language="console"> {
`mkdir -p ~/.local/xPacks/openocd
cd ~/.local/xPacks/openocd

tar xvf ~/Downloads/xpack-openocd-${ props.upstreamVersion }-${ props.xpackSubversion }-darwin-x64.tar.gz
chmod -R -w xpack-openocd-${ props.upstreamVersion }-${ props.xpackSubversion }`
}</CodeBlock>

On macOS, JTAG probes implemented as USB devices usually are
recognised directly and do not need explicit drivers.
