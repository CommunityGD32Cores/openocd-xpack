### Download

The macOS versions of **xPack OpenOCD**
are packed as `.tar.gz` archives.
Download the latest version named like:

* <code>xpack-openocd-{ props.version }-{ props.xpack_subversion }-darwin-x64.tar.gz</code>
* <code>xpack-openocd-{ props.version }-{ props.xpack_subversion }-darwin-arm64.tar.gz</code>

:::note

In case you wonder where the suffix comes
from, it is exactly the Node.js `process.platform` and `process.arch`.

:::

### Unpack

To manually install the <b>xPack OpenOCD</b>,
unpack the archive and move it to <code>~/.local/xPacks/openocd/xpack-openocd-{ props.version }-{ props.xpack_subversion }</code>:

<CodeBlock language="console"> {
`mkdir -p ~/.local/xPacks/openocd
cd ~/.local/xPacks/openocd

tar xvf ~/Downloads/xpack-openocd-${ props.version }-${ props.xpack_subversion }-darwin-x64.tar.gz
chmod -R -w xpack-openocd-${ props.version }-${ props.xpack_subversion }`
}</CodeBlock>

recognised directly and do not need explicit drivers.
