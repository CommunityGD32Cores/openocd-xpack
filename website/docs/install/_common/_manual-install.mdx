## Manual install

For all platforms, the **xPack OpenOCD** binaries are released as portable
archives that can be installed in any location.

The archives can be downloaded from
the <a href={ `https://github.com/xpack-dev-tools/openocd-xpack/releases/` }>GitHub Releases</a> pages.
